Why did I build this site?

I always want to make a personal website, and I have tried many kind blogger websites like Blogger and WordPress, but sadly them are all not very easy to use:

It is very hard to change the theme on those websites. I know front-end programming very well, and I really want to customize the theme by myself.

I really donâ€™t like write HTML code. When I tried to write something in good format on those websites, it turned out I have to write a bunch of HTML code which was tedious.

How did I build this site?

Used GitHub Pages to host my website. Though it can only host static web pages, I think it is good enough to host a demostration home page.

Utilized Jekyll to generate static web pages. I stoped using HTML + CSS + JavaScript model, since it is time-consuming. With Jekyll, I can write most of my site in Markdown. Since I am a minimalist, I started with a blank Jekyll framework, and customized theme all by myself.

Explored BootStrap 4 and SCSS to customize a neat and clean website style.
