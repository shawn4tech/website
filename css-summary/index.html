CSS Summary

1. How to use it in HTML file

1.1 内联方式

<h1 style=""></h1>

1.2 内部样式表

<head>
  <style type="text/css">
    h1 {...}
  </style>
</head>

1.3 外部样式表

  <head>
    <link rel="stylesheet" type="text/css" href="style.css" />
  </head>

2. Features:

  • 继承性
  • 层叠性
  • 优先级

10. Position

10.1 普通流定位

块元素由上到下，行元素由左到右

10.2 浮动定位

  • 将元素排除普通流
  • 元素依旧在包含框内
  • 浮动到左边或右边，直到碰到包含框或其他浮动元素
  • 多个块元素可能排列一行

float: none/left/right/inherit

clear: none/left/right/both/inherit

定义元素哪边不允许出现浮动元素（会跑到浮动元素下面）

10.3 相对定位/绝对定位/固定定位

10.3.1 定位属性

普通流/相对定位/绝对定位/固定定位
position: static/relative/absolute/fixed/sticky
偏移属性(必须在position确定时使用)：top, bottom, left, right: val

float/clear: 浮动定位

辅助：

  • z-index: 堆叠顺序

  • static: default
    Static positioned elements are not affected by the top, bottom, left, and right properties.

  • 相对定位：原本所在位置被保留
    An element with position: relative; is positioned relative to its normal position.

  • 绝对定位：
    • An element with position: absolute; is positioned relative to the nearest positioned ancestor (instead of positioned relative to the viewport, like fixed).
    • However; if an absolute positioned element has no positioned ancestors, it uses the document body, and moves along with page scrolling.
    • Note: A “positioned” element is one whose position is anything except static.
    • 从普通流移除
    • 偏移属性，相对于父元素
    • 不再占用原有位置
  • 固定定位：
    • An element with position: fixed; is positioned relative to the viewport
    • 从普通流移除
    • 不随页面滚动而移动
  • Sticky:
    • An element with position: sticky; is positioned based on the user’s scroll position.
    • A sticky element toggles between relative and fixed, depending on the scroll position. It is positioned relative until a given offset position is met in the viewport - then it “sticks” in place (like position:fixed).

      Note: Internet Explorer, Edge 15 and earlier versions do not support sticky positioning. Safari requires a -webkit- prefix (see example below). You must also specify at least one of top, right, bottom or left for sticky positioning to work.

  • z-index：

    • 解决重叠问题，设置优先级
    • 数字越大，优先级越高，离用户越近，可以为负数
