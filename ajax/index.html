AJAX and Usage

AJAX: Asynchonous Javascript And Xml

实质：使用XMLHttpRequest对象异步的向服务器发请求，服务器返回部分数据，而不是整个页面

1. 获取XMLHttpRequest对象

	// 现代浏览器都支持
	//IE5 IE6 使用ActiveXObject

	var xhr = null;
	if (window.XMLHttpRequest) {
		xhr = new XMLHttpRequest();
	} else {
		xhr = new ActiveXObject("Microsoft.XMLHttp");
	}

2. Attributes And Methods

  • Attributes
    • readyState: 请求的状态
      • 0: 未初始化
      • 1: 正在发送请求
      • 2: 发送请求完成
      • 3: 发送请求成功，正在接受数据
      • 4: 数据接受成功
    • onreadystatechange: 发生任何readyState变化时的事件控制对象
    • responseText:
    • responseXML: process it as DOM
    • status: HTTP status like 200, 202, 404 …
  • Methods:
    • abort();
    • getAllResponseHeaders();
    • getResponseHeader();
    • open(method, url);
    • send();
    • setRequestHeader();

2.1 发送请求的步骤

  1. 获取AJAX对象

  2. 设置回调函数 
     xhr.onreadystatechange = function() {
     if (xhr.readyState == 4 && xhr.status == 200) {
         ...;
     }
 };
  3. 创建请求 
     // GET
 xhr.open('get', 'list.do?name=value&name=value', true);
 /*
     true: 异步请求，用户可对页面操作
     false: 同步请求，页面锁死
 */

 // POST
 xhr.open('post', 'list.do', true);
 xhr.setRequestHeader('content-type', 'application/x-www-form-urlencoded');
 //post body 里面有内容，xhr默认不会主动加content-type
  4. 发送请求 
     // GET
 xhr.send(null);
 // POST
 xhr.send(name=value & name=value...);

3. Encoding

3.1 POST请求乱码

浏览器AJAX对象默认使用UTF-8 服务器不一定，默认iso-8859-1

解决方式：

req.setCharacterEncoding("utf-8");

3.2 GET乱码

IE AJAX use GBK (for Chinese browsers), others use UTF-8 服务器不一定，默认iso-8859-1

  • Solution 1: Tomcat: conf/server.xml 
      <Connector URIEncoding="utf-8">
  • Solution 2: 发送时指定 
      var uri = 'list.do?name=value&name=value';
  xhr.open('get', encodeURI(uri), true);

4. JSON

JSON: JavaScript Object Notation

Two structure:

  • Array : [val, val]
  • Collection: {key: val, key: val}

4.1 交换原理

	Server 			<----------->          Browser
Java Object ----> JSON String 		JSON String ---> JavaScript Object

4.1.1 JSON String —> JavaScript Object:

  • 使用JS原生函数，eval()
  • 有风险 
    var str = '{"name": "Sam", "age": 20}';
var obj = eval("(" + str + ")");
  • 使用原生JSON对象的parse()
  • 老版本不支持JSON对象 
    var str = '{"name": "Sam", "age": 20}';
var obj = JSON.parse(str);
  • 使用JSON.js 文件String的方法 
    var str = '{"name": "Sam", "age": 20}';
var obj = str.parseJSON();

4.1.2 JavaScript Object —> JSON String:

	var obj = {"name": "Sam", "age": 20};
	var str = JSON.stringify(obj);

	// 或者使用JSON.js提高的方法
	var str = obj.toJSONString();

4.1.3 Java Object —-> JSON String

JSON 官方提高的API, JSONObject, JSONArray

	Employee emp = new Employee(...);
	JSONObject obj = JSONObject.fromObject(emp);
	String jsonStr = obj.toString();

4.1.4 JSON String —-> Java Object

	String str = "{\"name\": \"Sam\", \"age\": 20}";
	JSONObject jsonObj = JSONObject.fromObject(str);
	Employee emp = (Employee)JSONObject.toBean(jsonObj, Employee.class);

	List<Employee> emps = (List<Employee>)JSONObject.toCollection(jsonArr, Employee.class);
